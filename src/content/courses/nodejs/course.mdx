Node.js isn't a language — it's V8 plus libuv plus a small standard library. This course is
the one we wish we had when we stopped using Node for toy scripts and started shipping real
servers. Every chapter is built around a runtime mental model — the event loop phases,
streams backpressure, observability budgets — and walks through the APIs that fall out of it.

Each chapter ships as three views: **Full Notes** for the deep read, **Quick Revision** for
pre-interview skimming, and a **Flow Diagram** so you can see the moving parts.
