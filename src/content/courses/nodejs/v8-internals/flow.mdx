## JavaScript on the Server — from request to CPU

```mermaid
flowchart TD
    subgraph NET [Network loop]
        C["<b>i · Client</b><br/>Browser / app<br/>knows server IP"]

        S -->|"RESPONSE<br/>HTML · JSON · bytes"| C
```
