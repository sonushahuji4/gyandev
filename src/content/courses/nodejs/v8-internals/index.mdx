## What V8 actually is

V8 is the JavaScript engine Node.js embeds. It takes source text and turns it into machine
code through a pipeline: parser → bytecode (Ignition) → optimized native code (TurboFan)
for hot paths.

## Why you care

You don't need to read V8's source to write Node well. But knowing that hidden classes
exist, that property access has inline caches, and that optimized code can be deoptimized
by shape changes — that's what separates "Node works" from "Node stays fast under load".
