## Problem Statement

There are `n` flights labelled `1 ‚Ä¶ n`. Given a list of bookings where each booking `[first, last, seats]` reserves `seats` seats on every flight in `[first, last]`, return an array `answer[i]` = total seats reserved on flight `i`.

## Input

- `bookings` ‚Äî list of `[first, last, seats]` triples (1-indexed inclusive range).
- `n` ‚Äî total number of flights.

```text title="Example input"
bookings = [[1,2,10],[2,3,20],[2,5,25]]
n        = 5
```

## Output

An array of length `n` where `answer[i]` = total seats reserved on flight `i + 1`.

```text title="Expected output"
[10, 55, 45, 25, 25]
```

## Constraints

- `1 ‚â§ n ‚â§ 2¬∑10‚Å¥`
- `1 ‚â§ bookings.length ‚â§ 2¬∑10‚Å¥`
- `1 ‚â§ first·µ¢ ‚â§ last·µ¢ ‚â§ n`
- `1 ‚â§ seats·µ¢ ‚â§ 10‚Å¥`

## Intuition

Na√Øvely adding `seats` to every index in `[first, last]` is `O(range)` per booking. A **difference array** converts each update into exactly two `O(1)` writes: `diff[l] += v` and `diff[r+1] -= v`. Prefix-summing the difference array reconstructs the actual values.

```text title="Walkthrough ‚Äî bookings above, n = 5"
diff = [0, 0, 0, 0, 0, 0]

After [1,2,10]:  [+10,   0, ‚àí10,   0,   0,   0]
After [2,3,20]:  [+10, +20, ‚àí10, ‚àí20,   0,   0]
After [2,5,25]:  [+10, +45, ‚àí10, ‚àí20,   0, ‚àí25]

Prefix sum:      [ 10,  55,  45,  25,  25]
                   ‚Üë                    ‚Üë
                  flight 1          flight 5
```

## Solution

<CodeTabs tabs={["Python", "TypeScript"]}>
<Fragment slot="tab-0">

```python title="corp_flight_bookings.py"
def corpFlightBookings(bookings: list[list[int]], n: int) -> list[int]:
    diff = [0] * (n + 1)
    for first, last, seats in bookings:
        diff[first - 1] += seats
        diff[last] -= seats

    result, running = [], 0
    for i in range(n):
        running += diff[i]
        result.append(running)

    return result
```

```ts title="corp_flight_bookings.ts"
function corpFlightBookings(bookings: number[][], n: number): number[] {
    const diff = new Array(n + 1).fill(0);
    for (const [first, last, seats] of bookings) {
        diff[first - 1] += seats;
        diff[last] -= seats;
    }

    const result: number[] = [];
    let running = 0;
    for (let i = 0; i < n; i++) {
        running += diff[i];
        result.push(running);
    }

    return result;
}
```

## Complexity

| Metric | Value | Reason |
| --- | --- | --- |
| Time | `O(n + m)` | `m` updates in `O(1)` each, plus one linear prefix-sum pass |
| Space | `O(n)` | Fixed-size difference array |

## Edge Case Thinking

| Scenario | Handling |
| --- | --- |
| Single booking covering all flights | `diff[0] += seats` and `diff[n] -= seats`; prefix sum gives a flat `[seats, ‚Ä¶]`. |
| Overlapping bookings | Combines additively ‚Äî exactly what the prefix sum produces. |
| Last booking reaches flight `n` | `diff[last] -= seats` is still a valid index because `diff` is allocated with size `n + 1`. |
| Bounded vs unbounded coordinates | Difference array is ideal when coords are bounded; fall back to sweep line for sparse/huge coordinates. |
| Empty `bookings` | Return `[0, 0, ‚Ä¶]` of length `n`. |
