## LeetCode #57 — Insert Interval

**Frequency:** 🔥 High. **Companies:** Google · Meta · LinkedIn.

## Problem Statement

Given a sorted list of non-overlapping intervals, insert a new interval. Merge if necessary. Return the result still sorted and non-overlapping.

```text title="Example"
Input:  intervals   = [[1,2],[3,5],[6,7],[8,10],[12,16]]
        newInterval = [4,8]
Output: [[1,2],[3,10],[12,16]]
```

## Intuition

Three-phase sweep:

1. Collect all intervals that end before the new one starts — no overlap, keep as-is.
2. Merge all intervals that overlap with the new one — expand the new interval.
3. Append remaining intervals after the merged zone.

No sorting needed since input is pre-sorted!

## Visual Walkthrough

```text
intervals: [1,2]  [3,5]  [6,7]  [8,10]  [12,16]
new:                [====4========8====]

Phase 1 — Before: [1,2] ends at 2 < 4 → keep [1,2]
Phase 2 — Merge:
  [3,5]  starts 3 ≤ 8 → merge → new = [3, 8]
  [6,7]  starts 6 ≤ 8 → merge → new = [3, 8]
  [8,10] starts 8 ≤ 8 → merge → new = [3, 10]
  → add [3, 10]
Phase 3 — After: [12,16] → keep [12,16]

Result: [[1,2], [3,10], [12,16]]
```

## Solution

<CodeTabs tabs={["Python", "TypeScript"]}>
<Fragment slot="tab-0">

```python title="insert_interval.py"
def insert(intervals: list[list[int]], newInterval: list[int]) -> list[list[int]]:
    result = []
    i, n = 0, len(intervals)

    # Phase 1: intervals entirely before newInterval
    while i < n and intervals[i][1] < newInterval[0]:
        result.append(intervals[i])
        i += 1

    # Phase 2: merge overlapping intervals
    while i < n and intervals[i][0] <= newInterval[1]:
        newInterval[0] = min(newInterval[0], intervals[i][0])
        newInterval[1] = max(newInterval[1], intervals[i][1])
        i += 1
    result.append(newInterval)

    # Phase 3: remaining intervals
    while i < n:
        result.append(intervals[i])
        i += 1

    return result
```

```ts title="insert_interval.ts"
function insert(intervals: number[][], newInterval: number[]): number[][] {
    const result: number[][] = [];
    let i = 0;
    const n = intervals.length;

    // Phase 1: before
    while (i < n && intervals[i][1] < newInterval[0]) {
        result.push(intervals[i]);
        i++;
    }

    // Phase 2: merge
    while (i < n && intervals[i][0] <= newInterval[1]) {
        newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
        newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
        i++;
    }
    result.push(newInterval);

    // Phase 3: after
    while (i < n) {
        result.push(intervals[i]);
        i++;
    }

    return result;
}
```

## Edge Case Thinking

<Callout type="tip" title="Edge cases">
  - **New interval before everything:** `[0,0]` into `[[1,5]]` → Phase 1 does nothing, Phase 2 does nothing, result = `[[0,0],[1,5]]`.
  - **New interval after everything:** Phase 1 adds all, Phase 2 adds new at end.
  - **New interval swallows everything:** Phase 2 merges all existing intervals into one giant merged interval.
  - **Empty intervals list:** Just return `[newInterval]`.
</Callout>
