JavaScript is the most misunderstood popular language. It looks simple — a
script for buttons — but under the hood it is a two-phase execution model,
a LIFO call stack, and a single thread juggling everything through the
event loop. This course is the one we wish we had when `var` first printed
`undefined` instead of crashing.

Each chapter ships as three views: **Full Notes** for the deep read,
**Quick Revision** for pre-interview skimming, and a **Flow Diagram** so
you can see the moving parts at a glance.
